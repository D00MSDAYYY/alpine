# Alpine — высокопроизводительная система визуализации данных Modbus/Tango

Alpine — это десктопное приложение на PySide6 для отображения графиков в реальном времени, получаемых от устройств через протоколы **Modbus TCP** и **Tango Controls**. Приложение оптимизировано для работы с большими потоками данных (до сотен точек в секунду) и поддерживает масштабирование временного окна, автоматическое обновление графиков и гибкую настройку внешнего вида.

## Возможности

- ✅ Потоковая отрисовка графиков с частотой до 100 Гц (без потери отзывчивости UI)
- ✅ Автоматическое и ручное масштабирование (auto-range)
- ✅ Остановка/возобновление обновления графиков
- ✅ Сохранение и загрузка настроек каналов (JSON)
